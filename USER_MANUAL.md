## 1. 登录后台

打开后台地址，输入管理员账号和密码，点击「登录」即可进入。

登录成功后会自动跳转到机器人管理页面。如果长时间没有操作，系统会自动退出登录，重新登录即可。

右上角显示当前登录的管理员名称，点击旁边的「退出」按钮可以退出登录。

## 2. 机器人管理

**位置：** 左侧菜单 →「机器人管理」

这里管理你所有的 Telegram 机器人。

### 添加机器人

1. 点击右上角「新增机器人」
2. 填写机器人名称（方便你自己辨认，随便起）
3. 填写机器人 Token（从 Telegram 的 @BotFather 获取）
4. 点击确定

### 验证 Token

添加完成后，建议点一下该机器人那行的「验证」按钮，系统会自动连接 Telegram 检查这个 Token 是否有效。验证通过后会自动填入机器人的 @用户名。

### 启用 / 停用

每个机器人有一个开关，打开表示该机器人正在运行，关闭则暂停服务。停用后用户发消息给这个机器人不会有任何响应。

### 编辑和删除

- 点击「编辑」可以修改名称或更换 Token
- 点击「删除」会永久移除该机器人及其所有关联数据，请谨慎操作

## 3. 邀请链接

**位置：** 机器人管理页面 → 某个机器人的「管理链接」按钮

每个机器人可以创建多条邀请链接。不同的链接可以配置不同的预览内容和广告，也方便你追踪用户来源。

### 创建链接

1. 点击「新增链接」
2. 填写链接名称（方便辨认，比如"渠道A"、"测试链接"）
3. 填写链接代码（英文字母、数字，比如 `channelA`）
4. 点击确定

创建后，系统会生成一条完整的邀请链接，格式为：

```
https://t.me/你的机器人用户名?start=链接代码
```

点击链接旁边的复制按钮可以一键复制，方便分发。

### 编辑和删除

- 点击「编辑」可以修改名称或代码
- 点击「删除」会移除该链接及其关联的内容和广告配置

## 4. 资源管理

**位置：** 左侧菜单 →「资源管理」

资源是你要展示给用户的图片和视频素材。所有资源统一在这里上传和管理，之后再到「内容配置」和「广告配置」中引用它们。

### 资源分组

页面左侧是分组列表，你可以把资源按用途分类（比如"产品图"、"宣传视频"、"广告素材"等）。

- 点击「新增分组」创建新的分类
- 点击分组名称可以筛选只看该分组下的资源
- 点击「全部资源」查看所有资源

### 上传资源

1. 点击右上角「上传资源」
2. 选择资源类型：
   - **图片**：单张图片
   - **视频**：单个视频文件
   - **图片组**：多张图片打包成一组，用户会看到一个可以左右滑动的相册
3. 选择所属分组（可选）
4. 填写说明文字（可选，会显示在图片或视频下方）
5. 选择文件并上传

上传视频后，系统会自动提取视频的时长、尺寸信息，并截取第一帧作为封面图，无需手动处理。

### 删除资源

每张资源卡片上有删除按钮，确认后会永久删除该资源文件。如果该资源已被内容或广告引用，删除后对应的配置也会失效，请注意检查。

## 5. 内容配置

**位置：** 左侧菜单 →「内容配置」

这里决定用户通过某条邀请链接进入机器人后，会看到哪些内容、以什么顺序展示。

### 配置步骤

1. 先在顶部选择一个机器人
2. 再选择该机器人下的一条邀请链接
3. 选择完成后，下方会显示当前已配置的内容列表

### 添加内容

1. 点击「添加资源」按钮
2. 在弹出的窗口中，可以按分组筛选或搜索资源
3. 勾选你想要的资源，点击确认
4. 新添加的资源会出现在列表末尾

### 调整顺序

内容列表支持拖拽排序。按住某一项上下拖动，松开即可改变展示顺序。用户看到的内容会按照这个顺序依次展示。

### 保存

调整完成后，务必点击「保存」按钮。不点保存的话，你的修改不会生效。

## 6. 广告配置

**位置：** 左侧菜单 →「广告配置」

广告会在用户浏览完内容后展示。配置方式和内容配置类似，但多了一个「按钮」功能。

### 配置步骤

1. 顶部选择机器人，再选择邀请链接（和内容配置一样）
2. 点击「添加资源」选择要作为广告展示的资源
3. 拖拽调整广告的展示顺序

### 广告按钮

这是广告配置独有的功能。每条广告下方可以添加可点击的按钮，用户点击后会跳转到你指定的网址。

1. 点击某条广告项，展开按钮编辑区域
2. 点击「添加按钮」
3. 填写按钮上显示的文字（比如"立即购买"、"了解详情"）
4. 填写点击后跳转的网址（必须是完整的网址，以 `http://` 或 `https://` 开头）
5. 可以添加多个按钮

> ⚠️ 注意：按钮的网址必须是有效的完整网址。如果填写了无效的网址（比如少了 `https://`），广告将无法正常显示。

### 保存

同样，修改完成后必须点击「保存」才会生效。

## 7. 用户列表

**位置：** 左侧菜单 →「用户列表」

这里可以查看所有通过机器人进入的用户记录，只能查看，不能修改或删除。

### 查看信息

每个用户会显示以下信息：

- Telegram ID：用户在 Telegram 上的唯一编号
- 用户名：用户的 @用户名（如果有的话）
- 姓名：用户设置的显示名称
- 来源机器人：用户是通过哪个机器人进来的
- 来源链接：用户点击的是哪条邀请链接
- 首次访问时间
- 最近访问时间

### 筛选和搜索

- 顶部搜索框可以按用户名或 ID 搜索
- 可以按机器人筛选
- 选择机器人后，还可以进一步按邀请链接筛选

这个功能主要用来了解用户来源分布，以及确认某条链接是否有人在使用。

## 8. 统计报表

**位置：** 左侧菜单 →「统计报表」

提供三个维度的数据概览。

### 今日概览

页面顶部的三张卡片，分别显示：

- 今日新增用户数
- 累计总用户数
- 今日广告展示次数

### 趋势图

中间区域是一个柱状图，展示每天的新增用户数量变化趋势。

- 默认显示最近 7 天的数据
- 可以通过日期选择器自定义查看范围
- 鼠标悬停在柱子上可以看到当天的广告展示次数

### 链接明细

底部表格按每条邀请链接分别统计：

- 所属机器人
- 链接名称和代码
- 该链接的总用户数
- 该链接今日新增用户数
- 该链接的总广告展示次数

可以点击表头排序，方便找出效果最好的链接。

## 9. 系统设置

**位置：** 左侧菜单 →「系统设置」

这里配置一些全局生效的参数。

### 预览结束文案

用户看完所有内容后，机器人会发送一段结束语。在这里可以自定义这段文字的内容。

### 预览结束按钮

结束语下方可以附带可点击的按钮（和广告按钮类似）。每个按钮需要填写显示文字和跳转网址。可以添加多个按钮，也可以不添加。

比如你可以设置一个"加入频道"的按钮，引导用户在预览结束后关注你的 Telegram 频道。

### 广告展示时长

控制每条广告在用户面前停留多少秒后，才出现「下一条」按钮。数值范围是 1 到 60 秒。

设置得太短，用户可能还没看清广告就跳过了；设置得太长，用户体验会变差。建议根据广告内容的复杂程度来调整，一般 5-10 秒比较合适。


### 保存

修改后点击「保存」按钮生效。

## 10. 常见问题

### 机器人没有响应？

- 检查该机器人是否已启用（机器人管理页面的开关是否打开）
- 检查 Token 是否正确（点击「验证」按钮确认）
- 确认服务器上的机器人程序是否正常运行

### 用户点击链接后没有内容？

- 确认该邀请链接下已经配置了内容（内容配置页面）
- 确认配置后点击了「保存」

### 广告不显示？

- 确认该邀请链接下已经配置了广告（广告配置页面）
- 检查广告按钮的网址是否有效（必须以 `http://` 或 `https://` 开头的完整网址）
- 确认配置后点击了「保存」

### 视频发送失败？

- 单个视频文件不能超过 50MB
- 确保上传的是常见视频格式（如 MP4）
- 如果之前能发现在不能发了，可能是文件损坏，尝试重新上传

### 图片组只显示了部分图片？

- Telegram 的图片组最多支持 10 张图片，超出的部分不会显示
- 确保上传的都是图片文件，不要在图片组里混入视频
